# Paris Café & Bakery WebGIS

Spatial Distribution of Cafés and Bakeries in Central Paris, France

## About
- **Theme:** Café & Bakery Distribution
- **Area:** Central Paris (Arrondissements 1-8)
- **Technology:** HTML, CSS, JavaScript, Leaflet.js
- **Data Source:** OpenStreetMap via HOT OSM Export Tool

## Features
- Interactive map with zoom & pan
- Café and Bakery point layers
- Main Road and Pedestrian Path layers
- Popup attributes on click
- Layer toggle on/off
- Filter by Café / Bakery
